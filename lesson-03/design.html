The colors identified above will be used as follows:

Primary Color

Will be used for major heading text colors, as well as the background color for the header and footer.

Secondary Color

Will be used as an accent color, as well as a text color for text appearing on dark background colors (header, footer, etc.)

Accent Color

Primarily intended to be a hyperlink color on lighter-colored backgrounds, can also be used on smaller graphical elements like borders, horizontal rules, etc.

Text Color

his will be the primary text color when text appears on light-colored backgrounds. Will also be used for lesser headings like H4, H5, and H6.

Background Color

This is the primary background color for most of a page layout.

Typography

All type will be set as follows:

Heading 1: Balthazar

To be used in header for site title.

Heading 2: Alegreya SC 600

To be used as individual page title in body of page.

Heading 3: Alegreya SC 600

Heading 4: Alegreya SC 500

Heading 5: Alegreya SC 500
Heading 6: Alegreya SC 500

Headings H3 through H6 will be used as subheadings, etc. for outlining the information on individual pages as needed, although it will be rare and unlikely that headings will get to a level past H4.

Body text is set in Balthazar 400 with a slightly larger line-spacing to make text easy to read.
